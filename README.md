# 基于机器学习的文本情感系统 python1114

## 项目概述

本项目是基于Python开发的文本情感分析系统。该系统通过机器学习算法对文本进行情感分类，并提供文本管理功能。

## 技术栈

- **后端：** Python
- **数据库：** MySQL
- **前端：** （具体技术未提及）
- **开发工具：** PyCharm
- **Python版本：** 3.7

## 功能模块

### 文本分类模块

- 通过用户输入文本进行情感检测。
- 设计文本对话框，方便用户输入。
- 对输入的文本内容进行情感分析，并输出结论。

### 文本管理模块

- 记录并展示文本分类的详细内容。
- 以列表形式展现文本输入内容和机器判定结果。

## 系统角色

- 用户：可输入文本并获取情感分析结果。
- 系统：进行情感分析，管理文本数据。
